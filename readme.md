# 手机J2ME和FC的模拟器

1. 安卓J2ME模拟器推荐使用: [JL-Mod_0.87.1-universal.apk](j2me%E6%A8%A1%E6%8B%9F%E5%99%A8/JL-Mod_0.87.1-universal.apk)
相比 [J2ME_Loader-1.8.2-open-release.apk](j2me%E6%A8%A1%E6%8B%9F%E5%99%A8/J2ME_Loader-1.8.2-open-release.apk) 而言，内置C盘挂载到sdcard目录

2. 安卓FC模拟器推荐使用: [lemuroid-app-free-dynamic-release.apk](fc%E6%A8%A1%E6%8B%9F%E5%99%A8/lemuroid-app-free-dynamic-release.apk)
亲测兼容性强，多亲F25完美使用，其他模拟器多亲都没法安装。

3. 安装J2ME模拟器里装FC模拟器推荐 [Nescube_V3.2.jar](j2me%E5%BA%94%E7%94%A8/Nescube_V3.2.jar)
唯一缺陷不支持中文


# 推荐手机

多亲F25。玩j2me和nes游戏绝配。

# 特殊说明

lemuroid作为安卓的fc模拟器，比如设置里把rom目录设置好，不然找不到游戏的
